# Decision Scenarios Using FinSight Analytics

## Scenario 1: Low Activation Rate
If activation rate falls below 60%, we:
- Audit onboarding flow
- Add guided dashboard walkthrough
- Measure impact via A/B test

## Scenario 2: Premium Users Retain Better
If premium users show higher retention:
- Introduce time-bound premium trial
- Push forecast feature earlier

## Scenario 3: Rising Churn Risk
If >20% of businesses inactive for 30 days:
- Trigger CS outreach
- Offer onboarding refresh sessions
